AI-Driven Unified Grievance Redressal System is a centralized web application that allows citizens to submit and track complaints through a single portal.
The system uses Natural Language Processing (NLP) and Machine Learning to automatically classify complaints (text and voice input) and route them to the appropriate department. It also provides a department dashboard for monitoring and resolving cases.
